# Resume Site (Next.js + Tailwind + TypeScript)

Single-page online resume and portfolio with a case-study route for projects.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000`.

## Build for production

```bash
npm run build
npm run start
```

## Content management

All editable content lives in:

- `data/resume.ts`

Update text, links, projects, and metadata there.
